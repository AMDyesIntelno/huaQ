# ueditor

1. 查看版本

大部分文章提到的查看版本方法都是通过`help.js`中的`document.getElementById('version').innerHTML = parent.UE.version;`,在控制台中执行`console.log(UE.version)`来得到版本

但其实可以直接通过访问`ueditor.all.js`来得到当前版本号
